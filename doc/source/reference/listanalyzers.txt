.. _listanalyzers:

listanalyzers
*************


**Syntax:** :code:`listanalyzers()`

Shows names of all available analysis methods that can be used in conjunction with :ref:`analyzeparticles` command. See also :ref:`headers` command.

This command can be used in a distributed processing mode, but it does not participate in distributed processing.
